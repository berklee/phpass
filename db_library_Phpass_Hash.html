Package: PHPass\Hash

PHPass Hash Class

This class provides a simple API for working with the various hash adapters. When instantiated with no arguments, it creates an instance of \Phpass\Hash\Adapter\Blowfish, configured for 2^12 (or 4,096) iterations. It is also possible to configure the class to use HMAC to provide some extra security where needed.

hashPassword($password);

// Check a password
if ($phpassHash->checkPassword($password, $passwordHash)) {
    // ...
}
